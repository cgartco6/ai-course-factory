AI_COURSE_FACTORY
-----------------

This system generates:
- Courses (video/audio/text/images)
- Marketing assets (posters, shorts, reels)
- Landing pages for different audiences
- Automated weekly reports

Folder Overview:
- backend/: all scripts and AI agents
- frontend/: landing pages + dashboard
- media/: generated content (images, audio, video)
- automation_data/: daily/weekly data
- course_package/: course content package (auto-generated)

Instructions:
1. Create virtual environment: python -m venv venv
2. Activate: venv\Scripts\activate
3. Install dependencies: pip install -r requirements.txt
4. Run: run_windows.bat
5. Frontend: open frontend/landing/index.html in browser

Note: Media folders and course_package are populated automatically by backend scripts.
